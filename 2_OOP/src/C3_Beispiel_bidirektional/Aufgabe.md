# Bidirektionale Assoziation

## Aufgabe
In einer grafischen Oberfläche (Klasse Gui) soll
der Vorname und der Nachname des Schülers 
"Mike Müller" angezeigt werden, wenn der Button ´Ok´
geklickt wird. D.h. die Methode ´clickedButtonOK()´ 
wird aufgerufen. 


Das Programm soll die Drei-Schichten-Architektur erfüllen.

1. Entwerfen Sie ein Klassendiagramm mit Attributen, Methoden 
    und Konstruktoren.

2. Entwerfen Sie ein Sequenzdiagramm für das oben beschriebene
    Szenario.
    
3. Ergänzen Sie das Klassendiagramm um die notwendigen
    Assoziationen mit Rollennamen und Kardinalitäten.

4. Kopieren Sie die Klasse ´Gui´ aus dem Tauschverzeichnis in Ihr
    Eclipse Projekt und implementieren Sie alle notwendigen Klassen
    für das Szenario aus Aufgabe 2.
    
5. Löschen Sie den Schülernamen wieder, wenn der Button ´Cancel´
    geklickt wird.
