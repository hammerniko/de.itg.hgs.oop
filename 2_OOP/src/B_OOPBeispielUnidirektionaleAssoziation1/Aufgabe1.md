# Aufgabe

 Es sollen Schueler mit Namen, Vornamen erstellt werden können.
 Jeder Schüler ist genau in einer Schulklasse. Für eine Schulklasse
 gibt es eine ausführliche Bezeichnung und eine Abkürzung für
 den Stundenplan.
 
 Der Schüler soll Zugriff auf seine Klasse haben und den Klassennamen
 ausgeben können.
 
 1.) Entwerfen Sie das Klassendiagramm in UMLET
 
 2.) Ein Schueler s1 ruft seinen Klassennamen auf 
     und gibt in aus. Entwerfen Sie das Sequenzdiagramm

 3.) Wie nennt man eine solche Beziehung zwischen 
     Klassen/Objekten. -> Assoziation (unidirektional) mit der Kardinalität 1
     
 4.) Implementieren Sie die Klassen Schueler und Schulklasse
 
 5.) Implementieren Sie das Szenario aus A2 in einer ausführbaren Startklasse.
